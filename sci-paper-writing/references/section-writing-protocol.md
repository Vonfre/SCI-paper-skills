# Section Writing Protocol

## Title Options

Generate multiple title types when useful:

- Descriptive: `X is associated with Y in Z`.
- Mechanistic: `X regulates Y through Z`.
- Translational: `X predicts Y and identifies Z`.
- Method/resource: `A [method/resource] for [task] in [system]`.

Use the target journal's observed title pattern only when supported by benchmark papers.

## Abstract Template

For unstructured abstracts:

1. Context and gap.
2. Objective or hypothesis.
3. Study design and methods.
4. Primary results with supplied quantitative evidence.
5. Interpretation and journal-relevant implication.

For structured abstracts, follow the journal headings exactly.

## Introduction Move Set

1. Field problem with journal-relevant stakes.
2. Current knowledge.
3. Specific unresolved gap.
4. Why existing approaches are insufficient.
5. Study objective and design.
6. Contribution sentence.

## Results Move Set

Each result subsection:

1. Purpose sentence.
2. Method/comparison.
3. Main observation.
4. Quantitative/statistical evidence.
5. Figure/table/supplement citation.
6. Bridge to the next result.

## Discussion Move Set

1. Principal findings.
2. Comparison with prior literature.
3. Mechanistic/clinical/methodological implications.
4. Strengths.
5. Limitations.
6. Future work.
7. Evidence-matched conclusion.

## Language Rules

- Use precise verbs: reveal, indicate, suggest, demonstrate only when evidence supports them.
- Avoid inflated novelty: "first" requires verification.
- Avoid unsupported causality.
- Prefer concise transitions that clarify logic, not decorative phrasing.
- Keep placeholders visible until the user supplies facts.
