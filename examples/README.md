# Examples

This directory shows how to use `SCI-paper-skills` as a connected manuscript workflow, not as isolated prompts.

## Start Here

- `zero-to-one-sci-manuscript/complete-manuscript.md`: a complete English SCI-style paper generated by the skill workflow.
- `zero-to-one-sci-manuscript/complete-manuscript.zh-CN.md`: a complete Chinese SCI-style paper generated by the same workflow.
- `zero-to-one-sci-manuscript/complete-manuscript.docx`: a checked English Word manuscript generated from the example.
- `zero-to-one-sci-manuscript/complete-manuscript.zh-CN.docx`: a checked Chinese Word manuscript generated from the example.
- `zero-to-one-sci-manuscript/`: supporting files showing the initial brief, stage trace, final package metadata, and state snapshot.
- `manuscript-state-example.yaml`: the final shared `manuscript_state` snapshot produced by the zero-to-one example.

Use examples as workflow references, not as scientific facts. Real projects must replace all example claims, sources, figures, journal details, statistics, and citations with verified user data and current literature.
