# Examples

This directory contains small examples that show how the tight workflow should carry state across skills.

## Files

- `manuscript-state-example.yaml`: a compact example of the shared `manuscript_state` object with linked claim, figure, source, section, reviewer, citation, and submission fields.

Use examples as shape references, not as scientific facts. Real projects must replace all example claims, sources, figures, and journal details with verified user data and current literature.
